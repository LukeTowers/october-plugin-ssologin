How to configure

  1. Go to https://console.developers.google.com/project
  2. Create a new project and call it whatever you want (ex: OctoberOAUTH)
  3. Once the project was created, click on APIs & Auth > APIs > API Library and enable Google+ API
  4. Next, go to APIs & Auth > Credentials > Add credentials and select OAuth 2.0 client ID (configure "consent screen" if necessary)
  5. On "Create client ID" screen, select "Web application"
  6. Fill:
    • Authorized JavaScript origins: (or you can leave this field blank if you want)
    • Authorized redirect URIs: /martin/ssologin/google
  7. Click Create; a popup will appear with the "client ID" and "client secret", please, copy and paste in the field below (now you can close Google Developer Console window)
  8. Save this form, and you're ready to login with your Google account.
